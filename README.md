# AI Sales Analytics Dashboard

A Streamlit-based AI-powered Sales Analytics Dashboard with:
- User Authentication
- Role-Based Admin Panel
- Interactive Charts using Plotly
- AI Chatbot Integration (Google Gemini + Groq)
- Sales & Profit Analysis
- User Management System

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Google Gemini API
- Groq API

## Features
- Login & Signup System
- Admin/User Roles
- Interactive Dashboard
- AI Startup Analyzer
- Advanced Analytics

## Run Project
```bash
pip install -r requirements.txt
streamlit run 1.py
